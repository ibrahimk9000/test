## Build



### Windows 

- viual studio 2015 or better
- extract SFML archive ($projectdir/sfml)
- build project

### Linux

- install SFML
```shell
$ sudo apt-get install libsfml-dev (debian based)
```
 

- build makefile
```shell
$ make
```
